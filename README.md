# Aurora-Extension-UI-for-Chrome
Chrome Custom theme extension
Steps to Enable the Extension on Chrome
Download the Zip file from the link
And extract the zip file contents where you won't move/delete 
Open Chrome and Navigate to Extensions
And Click on Manage Extensions 
Enable Developer Mode
And Select the "Load Unpacked" option after enabling Developer mode
select the extracted folder, Open a New tab in chrome, click on Keep it and start customizing
